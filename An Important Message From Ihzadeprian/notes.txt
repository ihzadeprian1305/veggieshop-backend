Website ini dibangun untuk menyelesaikan tugas Pemrograman Mobile dengan menggunakan Laravel untuk membangun API dan Kotlin untuk Aplikasi Mobile yang dibangun

Yang meliputi materi CRUD dengan menggunakan Database dan menggunakan framework Laravel 8
Bertemakan Aplikasi Mobile pembelian sayur di daerah Kota Malang

Menggunakan Local Server Laragon dengan menjalankan perintah 'php artisan serve' dan mengakses http://127.0.0.1:8000/

Database berada pada file veggieshop-backend.sql jika ingin yang sudah terdapat banyak data
Atau bisa dengan perintah 'php artisan migrate --seed' namun isi Database kosong

*Saya sering membuat project Aplikasi Mobile menggunakan Flutter dan Laravel yang sering saya gunakan dalam membangun API dari Aplikasi Mobile tersebut
